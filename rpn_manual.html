RPN计算器使用手册

什么是逆波兰表达式？

逆波兰表达式（Reverse Polish Notation, RPN）是一种数学表达式的表示方法，操作符位于操作数之后。

传统表达式： 3 + 4
逆波兰表达式： 3 4 +

安装和编译

# 克隆项目
git clone https://github.com/xiaodan-1213/rpn_calculator.git
cd rpn_calculator

# 编译项目
make

# 运行计算器
./rpn_calculator

基本使用方法

输入表达式: 5 2 +
结果: 7

输入表达式: 3 4 * 2 /
结果: 6

输入表达式: 10 5 - 2 *
结果: 10

支持的运算符

  • + - 加法
  • - - 减法
  • * - 乘法
  • / - 除法

复杂表达式示例

# 计算 (5 + (1 + 2) * 4) - 3
输入表达式: 5 1 2 + 4 * + 3 -
结果: 14

# 计算 (10 - 2) * 3
输入表达式: 10 2 - 3 *
结果: 24

错误处理

栈下溢错误： 操作数不足时会出现此错误
无效操作符错误： 输入不支持的操作符时会出现此错误

退出程序

  • 按 Ctrl + D 正常退出
  • 按 Ctrl + C 强制退出

项目信息
